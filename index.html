Mirai

Mirai is an Discord music bot written for Python 3.5+, using the discord.py library. Learn more

It plays requested songs, from YouTube and other services, into a Discord server (or multiple servers) and if the queue becomes empty it will play through a list of existing songs, if configured to do so. Mirai features a permissions system allowing owners to restrict commands to certain people. As well as playing songs, Mirai is capable of streaming live media into a voice channel (experimental).
